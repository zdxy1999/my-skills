# 八字命理姻缘匹配 Skill

基于传统八字命理的姻缘匹配分析工具，用于根据个人八字推荐合适的伴侣配置。

## 特点

- **零依赖**：不需要任何外部运行环境
- **知识驱动**：基于传统命理规则和Claude推理能力
- **标准化输出**：JSON格式的匹配结果
- **可解释性**：每个推荐都有明确的命理依据

## 使用方法

### 方式1：提供生日+性别

```json
{
  "birth_date": "1990-09-24",
  "birth_time": "20:11",
  "gender": "男",
  "age_range": {"min": -3, "max": 5}
}
```

### 方式2：直接提供八字

```json
{
  "bazi": "己卯 癸酉 己卯 甲戌",
  "gender": "男",
  "age_range": {"min": -3, "max": 5}
}
```

## 输出格式

返回JSON格式的匹配结果，包括：

1. **用户八字分析**：日主、五行分布、配偶星、夫妻宫等
2. **匹配候选人列表**：按评分排序的推荐，包含匹配理由和注意事项
3. **总结统计**：总体分析结果和建议

## 核心原理

### 五行互补

根据用户八字的五行强弱，寻找能够互补的伴侣：

- 身弱需要帮扶五行（印比）
- 身强需要泄耗五行（食伤财）
- 缺失五行需要补充

### 配偶星匹配

- 男命看财星（正财、偏财）
- 女命看官杀星（正官、七杀）
- 配偶星出现且有力者更佳

### 夫妻宫分析

- 日支为夫妻宫，代表配偶和家庭
- 查看坐什么十神，判断配偶特质
- 避免与日支形成六冲、六害

### 评分标准

总分100分：
- 五行互补性（40分）
- 配偶星匹配（30分）
- 夫妻宫稳定（20分）
- 整体和谐（10分）
