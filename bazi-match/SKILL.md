# 八字命理姻缘匹配 Skill

## When to Use

当用户提出以下需求时，使用此技能：

- **八字姻缘分析**：分析个人的姻缘运势、婚姻特点
- **命理配对**：比较两个人八字的匹配程度
- **伴侣推荐**：根据个人八字推荐合适的伴侣类型
- **八字合婚**：传统意义的八字合婚分析
- **婚姻咨询**：基于命理的婚姻建议

触发关键词示例：
- "八字配对"
- "姻缘分析"
- "八字合婚"
- "适合什么样的伴侣"
- "帮我看看八字姻缘"

## 工作流程

### Step 1: 接收输入信息

支持两种输入方式：

#### 方式1：生日+性别（推荐）

用户提供：
- 出生日期（公历，如：1990-09-24）
- 出生时间（如：20:11，可选，不知时可忽略）
- 性别（男/女）
- 年龄范围偏好（可选，如：±3岁）
- 性取向（可选，默认异性恋）

```json
{
  "birth_date": "1990-09-24",
  "birth_time": "20:11",
  "gender": "男",
  "age_range": {"min": -3, "max": 5},
  "sexual_orientation": "heterosexual"
}
```

#### 方式2：直接八字

用户提供：
- 八字四柱（如：己卯 癸酉 己卯 甲戌）
- 性别
- 年龄范围偏好（可选）

```json
{
  "bazi": "己卯 癸酉 己卯 甲戌",
  "gender": "男",
  "age_range": {"min": -3, "max": 5}
}
```

### Step 2: 解析用户八字

当用户提供生日时，需要推算八字：

1. **确定四柱**：年柱、月柱、日柱、时柱
2. **识别日主**：日干（代表命主自己）
3. **统计五行**：计算金木水火土的数量和强弱
4. **定位关键要素**：
   - 日支 = 夫妻宫（代表配偶和家庭）
   - 男命的财星 = 妻星（正财=正妻，偏财=情人/偏缘）
   - 女命的官杀星 = 夫星（正官=正夫，七杀=偏缘）

### Step 3: 分析命局特征

#### 3.1 日主旺衰判断

判断日主强弱，主要看三个因素：

- **得令**：日主生于旺月（得月令之气）
  - 木旺于春（寅卯辰月）
  - 火旺于夏（巳午未月）
  - 金旺于秋（申酉戌月）
  - 水旺于冬（亥子丑月）
  - 土旺于四季月（辰戌丑未月）

- **得势**：得众多同类五行帮扶
  - 天干透出比劫（同五行）
  - 地支有强根

- **得地**：日支坐下有根气

**旺衰判定**：
- 得令+得势+得地 = 身旺
- 不得令+失势+失地 = 身弱
- 中间状态为中和

#### 3.2 姻缘关键要素分析

1. **配偶星分析**
   - 是否出现（透干或藏支）
   - 强弱程度（得令、有根否）
   - 位置（年月日时）
   - 有无冲克（是否被合化、冲散）

2. **夫妻宫分析**
   - 坐什么十神（查看日支对应的十神）
   - 宫位吉凶（有无冲、害、空）
   - 宫内藏干

3. **五行平衡**
   - 哪些五行过强
   - 哪些五行缺失或过弱
   - 需要补充什么五行

4. **神煞影响**（辅助参考）
   - 桃花（子午卯酉）：异性缘好
   - 天乙贵人：得贵人助力
   - 红鸾：姻缘喜事
   - 羊刃、驿马：影响婚姻稳定

### Step 4: 确定匹配需求

根据以上分析，确定需要什么样的伴侣：

1. **五行互补**
   - 身弱需要补印比（生扶日主的五行）
   - 身强需要补食伤财（泄耗日主的五行）
   - 缺某五行需要补该五行

2. **配偶星需求**
   - 配偶星弱的需要配偶星强的
   - 无配偶星的需要有配偶星的

3. **冲克规避**
   - 避免与日支形成六冲（子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲）
   - 避免与日支形成六害（子未害、丑午害、寅亥害、卯辰害、酉戌害、申巳害）

### Step 5: 筛选和评分候选人

从候选人池中筛选符合基本条件的：

1. **基础筛选**
   - 符合性别和年龄要求
   - 无严重冲克关系

2. **评分计算**（总分100分）

   **a. 五行互补性（40分）**
   - 候选人五行能补充用户所需：30-40分
   - 部分互补：15-29分
   - 互补性弱：0-14分

   **b. 配偶星匹配（30分）**
   - 配偶星五行相生、位置得当：25-30分
   - 配偶星可用：15-24分
   - 配偶星弱或不合适：0-14分

   **c. 夫妻宫稳定（20分）**
   - 无冲克、宫位吉利：15-20分
   - 有轻微影响：8-14分
   - 有严重冲克：0-7分

   **d. 整体和谐（10分）**
   - 有六合、三合等吉神：8-10分
   - 中平：4-7分
   - 有凶神：0-3分

3. **排序推荐**
   - 按评分从高到低排序
   - 输出前5-10名推荐

## 核心匹配规则

### 五行生克基础

**相生**（促进、帮助）：
- 木生火
- 火生土
- 土生金
- 金生水
- 水生木

**相克**（制约、克制）：
- 木克土
- 土克水
- 水克火
- 火克金
- 金克木

### 配偶星识别

**男命**（日主为干）：
- 正财 = 正妻（异性五行，阴阳不同）
  - 如：甲木男，己土为正财
- 偏财 = 偏缘、情人（同性五行）
  - 如：甲木男，戊土为偏财

**女命**（日主为干）：
- 正官 = 正夫（克我者，阴阳不同）
  - 如：甲木女，辛金为正官
- 七杀 = 偏缘（克我者，同性）
  - 如：甲木女，庚金为七杀

### 夫妻宫分析

日支（日柱地支）为夫妻宫，查看坐什么十神：

1. **坐财星/官星**（吉）：配偶得力，婚姻和谐
2. **坐食伤**（中平）：配偶有才华，但可能自我中心
3. **坐比劫**（不利）：容易有竞争，感情不稳定
4. **坐印星**（吉）：配偶照顾自己，家庭温馨
5. **坐杀枭**（复杂）：对方能干但压力大

### 冲克合化关系

**六冲**（子午、丑未、寅申、卯酉、辰戌、巳亥）：
- 冲夫妻宫最不利
- 其他柱冲可减轻，但仍需注意

**六害**（子未、丑午、寅亥、卯辰、酉戌、申巳）：
- 影响感情深度
- 容易产生误会

**六合**（子丑、寅亥、卯戌、辰酉、巳申、午未）：
- 增进感情
- 有利婚姻

**三合**（申子辰、亥卯未、寅午戌、巳酉丑）：
- 三合局力量大
- 有利家庭和谐

4. **真实感情需要经营**，命理匹配不能保证幸福
