# 八字匹配规则详解

## 一、评分体系

匹配分析采用百分制评分，分为四个维度：

| 维度 | 权重 | 评分要点 |
|------|------|----------|
| 五行互补性 | 40分 | 五行能否相互补充、流通 |
| 配偶星匹配 | 30分 | 配偶星配置是否合理 |
| 夫妻宫稳定 | 20分 | 夫妻宫是否有冲克 |
| 整体和谐 | 10分 | 其他吉凶神煞影响 |

---

## 二、五行互补性评分（40分）

### 评分标准

#### 优秀（30-40分）

**条件**：
- 候选人的强五行正是用户所需的弱五行
- 能够形成完整的五行流通
- 候选人五行格局能完美补充用户缺失

**示例**：
- 用户：缺金水，木火旺
- 候选人：金水旺，火木弱
- 评分：38分

**评分细则**：
- 完美互补（缺的都有，弱的都补）：38-40分
- 高度互补（主要需求都满足）：32-37分
- 良好互补（大部分需求满足）：30-31分

#### 良好（15-29分）

**条件**：
- 部分五行互补
- 主要需求能部分满足
- 整体趋势相合

**示例**：
- 用户：缺金水
- 候选人：有金但水不足
- 评分：22分

**评分细则**：
- 部分互补较强：24-29分
- 中度互补：20-23分
- 基础互补：15-19分

#### 一般（0-14分）

**条件**：
- 互补性弱
- 只有个别五行能互补
- 整体格局冲突

**示例**：
- 用户：缺金水
- 候选人：火土旺，金水弱
- 评分：8分

**评分细则**：
- 少量互补：8-14分
- 几乎无互补：0-7分

### 互补性判断方法

#### 1. 日主强弱互补

**用户身弱**：需要印星（生我）和比劫（同我）

| 用户日主 | 需要的印星 | 需要的比劫 |
|----------|------------|------------|
| 甲乙木 | 水（壬癸） | 木（甲乙） |
| 丙丁火 | 木（甲乙） | 火（丙丁） |
| 戊己土 | 火（丙丁） | 土（戊己） |
| 庚辛金 | 土（戊己） | 金（庚辛） |
| 壬癸水 | 金（庚辛） | 水（壬癸） |

**用户身强**：需要食伤（我生）和财星（我克）

| 用户日主 | 需要的食伤 | 需要的财星 |
|----------|------------|------------|
| 甲乙木 | 火（丙丁） | 土（戊己） |
| 丙丁火 | 土（戊己） | 金（庚辛） |
| 戊己土 | 金（庚辛） | 水（壬癸） |
| 庚辛金 | 水（壬癸） | 木（甲乙） |
| 壬癸水 | 木（甲乙） | 火（丙丁） |

#### 2. 缺失五行补充

统计用户八字中缺失或极弱的五行，候选人有相应五行则加分。

#### 3. 五行流通

检查双方八字组合后，五行是否能流通：

```
示例：
用户：木旺 → 火弱 → 土弱 → 金弱 → 水
候选：水旺 → 木旺 → 火旺 → 土旺 → 金弱

组合后形成：水生木，木生火，火生土，土生金
流通：✓ 流畅
评分：加分
```

## 三、配偶星匹配评分（30分）

### 男命评分标准

#### 优秀（25-30分）

**条件**：
- 候选人（女）八字财星配置好
- 财星得令、有根、透干
- 财星五行与用户日主相生

**示例**：
- 用户：甲木日主
- 候选人：己土（正财）或戊土（偏财）旺，且有金生水
- 评分：28分

#### 良好（15-24分）

**条件**：
- 有财星但力量一般
- 财星配置可用
- 无严重冲克

**评分细则**：
- 财星较有力：20-24分
- 财星中等：15-19分

#### 一般（0-14分）

**条件**：
- 财星弱或缺失
- 财星被冲克
- 财星五行不合

### 女命评分标准

#### 优秀（25-30分）

**条件**：
- 候选人（男）八字官杀星配置好
- 官杀得令、有根、透干
- 官杀五行与用户日主有情

**示例**：
- 用户：甲木日主
- 候选人：辛金（正官）或庚金（七杀）旺，且有土生金
- 评分：28分

#### 良好（15-24分）

**条件**：
- 有官杀但力量一般
- 官杀配置可用
- 无严重冲克

#### 一般（0-14分）

**条件**：
- 官杀弱或缺失
- 官杀被冲克
- 官杀五行不合

### 配偶星质量判断

#### 财星质量（男命用）

| 质量 | 特征 | 分数 |
|------|------|------|
| 上等 | 得令+有根+透干+无冲 | 25-30分 |
| 中等 | 得令或 有根或 透干（满足其一） | 15-24分 |
| 下等 | 不得令+无根+不透干 | 0-14分 |

#### 官杀质量（女命用）

| 质量 | 特征 | 分数 |
|------|------|------|
| 上等 | 得令+有根+透干+无冲 | 25-30分 |
| 中等 | 得令或 有根或 透干（满足其一） | 15-24分 |
| 下等 | 不得令+无根+不透干 | 0-14分 |

## 四、夫妻宫稳定评分（20分）

### 评分标准

#### 优秀（15-20分）

**条件**：
- 用户日支与候选人日支无冲克
- 有六合、三合等吉神
- 双方日支五行相生有情

**示例**：
- 用户日支：子
- 候选人日支：丑（六合）或 申/辰（三合）
- 评分：18分

#### 良好（8-14分）

**条件**：
- 无明显冲克
- 无合化但不冲突
- 五行相容

**评分细则**：
- 五行相容但不合：10-14分
- 中平无碍：8-9分

#### 一般（0-7分）

**条件**：
- 有六冲
- 有六害
- 五行严重相克

**示例**：
- 用户日支：子
- 候选人日支：午（六冲）
- 评分：3分

### 地支关系速查

## 五、整体和谐评分（10分）

### 评分标准

#### 优秀（8-10分）

**条件**：
- 有天乙贵人、天德月德等吉神
- 桃花位置得当
- 无凶神干扰

**评分细则**：
- 吉神多：9-10分
- 有主要吉神：8分

#### 良好（4-7分）

**条件**：
- 无特别吉神也无凶神
- 整体中平
- 小问题可接受

**评分细则**：
- 小吉神：6-7分
- 中平：4-5分

#### 一般（0-3分）

**条件**：
- 有羊刃、亡神等凶神
- 桃花过多且混杂
- 多种不利因素

### 神煞影响评分

#### 吉神加分

| 神煞 | 加分 | 条件 |
|------|------|------|
| 天乙贵人 | +2 | 命局有贵人 |
| 桃花（适度） | +1 | 1-2个桃花 |
| 红鸾 | +2 | 姻缘吉利 |
| 文昌 | +1 | 聪明智慧 |

#### 凶神减分

| 神煞 | 减分 | 条件 |
|------|------|------|
| 羊刃 | -2 | 性格刚强易冲 |
| 桃花过多 | -2 | 3个以上桃花 |
| 亡神 | -1 | 精神困扰 |
| 驿马过多 | -1 | 不稳定 |

## 六、综合评分计算

### 计算公式

```
总分 = 五行互补性得分 + 配偶星匹配得分 + 夫妻宫稳定得分 + 整体和谐得分
```

### 等级划分

| 总分 | 等级 | 建议 |
|------|------|------|
| 85-100 | 高度匹配 | 强烈推荐，优先考虑 |
| 70-84 | 良好匹配 | 推荐交往，深入了解 |
| 55-69 | 中度匹配 | 可以尝试，需努力经营 |
| 40-54 | 一般匹配 | 谨慎考虑，需大量磨合 |
| 0-39 | 不太匹配 | 不建议，困难较大 |

## 七、特殊情况处理

### 1. 特殊格局

#### 从格

- **从旺格**：命局全是同五行，需从其强
- **从杀格**：七杀极旺，日主无条件从之
- **从财格**：财星极旺，日主无条件从之

特殊格局的匹配规则与普通格局不同，需单独分析。

#### 化气格

- **甲己合化土**：甲木与己土相遇，条件合适则化土
- **乙庚合化金**：乙木与庚金相遇，条件合适则化金
- **丙辛合化水**：丙火与辛金相遇，条件合适则化水
- **丁壬合化木**：丁火与壬水相遇，条件合适则化木
- **戊癸合化火**：戊土与癸水相遇，条件合适则化火

化气格需考虑化神的作用。

### 2. 冲克合化复杂

当多个地支关系复杂时：

1. **优先级**：合化 > 冲克 > 普通
2. **远近**：日柱关系 > 月柱关系 > 年柱关系
3. **强弱**：力量大的决定性强

### 3. 空亡

日支或时支空亡时：

- 空亡表示该宫位力量减弱
- 日支空亡：夫妻宫有缺失感
- 配偶星空亡：缘分较弱

## 八、推荐理由生成

### 匹配理由应包含

1. **五行互补**：具体说明哪几行互补
2. **配偶星优势**：配偶星的配置如何好
3. **夫妻宫稳定**：夫妻宫关系如何和谐
4. **整体和谐**：其他有利因素

### 注意事项应包含

1. **潜在问题**：可能的冲克或不利因素
2. **需要磨合**：性格差异等需要调整的地方
3. **经营建议**：如何维护感情

## 九、评分示例

### 示例1：高度匹配

**用户**：
- 八字：己卯 癸酉 己卯 甲戌
- 日主：己土（偏弱）
- 缺：金水弱

**候选人**：
- 八字：丁丑 己酉 辛酉 壬辰
- 日主：辛金
- 特点：金水旺

**评分**：
1. 五行互补：38/40（金水完美补充）
2. 配偶星匹配：27/30（财星有力）
3. 夫妻宫稳定：16/20（略有相克但可通关）
4. 整体和谐：8/10（有吉神）

**总分：89分（高度匹配）**

### 示例2：中度匹配

**用户**：
- 八字：甲子 丙寅 戊午 庚申
- 日主：戊土（身强）
- 需：金水

**候选人**：
- 八字：乙卯 己卯 丁亥 辛丑
- 日主：丁火
- 特点：木火旺

**评分**：
1. 五行互补：15/40（互补性弱）
2. 配偶星匹配：20/30（官星可用）
3. 夫妻宫稳定：12/20（无冲克）
4. 整体和谐：5/10（中平）

**总分：52分（一般匹配）**

## 十、使用建议

1. **综合考量**：不要只看总分，要看各维度
2. **关注短板**：低分维度是需要注意的地方
3. **理性看待**：命理只是参考，不是决定因素
4. **人为重要**：性格、价值观、人品更重要
5. **与时俱进**：现代婚姻观念与传统需结合

匹配分析的目的在于帮助了解自己和他人，而非宿命论。真正的幸福建立在相互理解、尊重和经营之上。
