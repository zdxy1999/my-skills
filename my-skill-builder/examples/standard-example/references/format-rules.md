# Format Rules Reference

## Markdown Rules
- Headers: Use ATX style (`## Header`)
- Lists: Use `-` for unordered, `1.` for ordered
- Code blocks: Use fenced blocks with \`\`\` language
- Links: Use `[text](url)` format
- Images: Use `![alt](url)` format

## Text Rules
- Line endings: LF only
- Encoding: UTF-8
- Newlines at EOF: Exactly one
- Trailing whitespace: None
- Max line length: 100 (soft limit)

## HTML Rules
- Indent: 2 spaces
- Self-closing: Use `<tag />`
- Attributes: Double quotes
